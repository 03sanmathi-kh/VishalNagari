Multi-Lane Road Separation + Smart Gate Control System

Problem Statement

All types of vehicles currently move on the same road:

  • EV buses
  • EV cars
  • EV scooters
  • Petrol/Diesel (Resident) Vehicles
  • Outsider Vehicles
  • Goods Trucks

This causes lane mixing, congestion, delays, and unsafe driving. The city needs **dedicated lanes** with **smart gates** that allow only correct vehicle types.

Solution

A fully automated lane management system that:

  • Reads vehicle number using ANPR
  • Classifies vehicle type using a hash table
  • Opens gate only for allowed vehicles
  • Uses BFS to check congestion status
  • Queues vehicles when lane is full
  • Continuously monitors all lanes in real time

Data Structures Used

  • Graph → Lane connectivity
  • Hash Table → Vehicle number → vehicle type
  • Queue → Vehicles waiting at lane

Algorithms Used

  • BFS → Checks if lane path is free or congested
  • Hash Lookup → Fast classification O(1)
  • Selection Sort → Sorts lane loads (optional)

Algorithm Efficiency

  • Hash Table: O(1)
  • BFS: O(V + E)
  • Queue: O(1) operations
  • Selection Sort: O(n²) — for lane load sorting

Step-by-Step Process

1. Vehicle arrives at lane gate → ANPR reads plate.

2. Hash table identifies vehicle type.

3. Based on type:

  • EV Car → EV-Car Lane
  • EV Bus → EV-Bus Lane
  • EV Scooter → EV-Scooter Lane
  • Resident Petrol/Diesel → Non-EV Lane
  • Outsider → Visitor Lane
  • Truck → Goods Lane

4. BFS checks lane congestion.

5. If lane free → gate opens.

6. If congested → vehicle joins queue.

7. Dashboard updates lane status continuously.
