Module 8: Smart Energy-Efficient Street Lighting System

Problem Statement

In emerging smart cities, streetlights often waste electricity by staying ON during low traffic periods or in sparsely populated zones. The challenge is to create a real-time, adaptive system that:
  • Monitors pedestrian, vehicle, and industrial activity across the city.
  • Activates only the necessary streetlights while maintaining safety.
  • Minimizes energy consumption without compromising coverage.
  • Adapts dynamically to sudden changes in city activity.

Algorithms Used

1. Queue-Based Event Simulation: Handles city activity events in order.
2. Union-Find (DSU): Clusters connected streetlight zones for optimized lighting.

Algorithm Efficiency

Queue: O(1) per event → perfect for real-time changes.

Union-Find: ≈O(1) with path compression → efficient cluster updates.

Design Techniques Used

Event-Driven Simulation: System reacts only when something changes.
Cluster Optimization: Avoids turning ON unnecessary lights.

How This Aligns with SDGs
