Ayushman Bharat – Hospital Appointment Optimization

This module focuses on optimizing hospital appointments and resource allocation for beneficiaries of the Ayushman Bharat scheme. The challenge is to minimize patient waiting times, efficiently allocate doctors and beds, and ensure timely care for high-priority patients.

Algorithm Used:

Greedy Algorithm with Priority Queue and Dynamic Programming is applied to assign patients to optimal time slots. Graph-based shortest path and reachability checks are optionally used to assign patients to the nearest available hospitals.

Algorithm Efficiency: O(n log n) for scheduling via priority queue; Dynamic Programming reduces repeated computations for time slot optimization
