Intelligent Traffic Density Management System

Problem

Traffic becomes highly congested on certain roads while others remain free. Manual monitoring fails during peak hours, causing delays.

Solution

A smart system that collects traffic density in real time, sorts it, identifies busy roads, and adjusts signals automatically.

Data Structures Used

  • Array ‚Üí stores density values
  • Max Heap ‚Üí pick highest congestion
  • Queue ‚Üí simulate waiting vehicles
  • Graph ‚Üí represent intersections

Algorithms Used

  • Merge Sort ‚Üí sort densities
  • Greedy Allocation ‚Üí assign green time
  • BFS ‚Üí network update

Step-by-Step Process

1. Sensors collect density values into an array.

2. Merge Sort arranges roads by density.

3. Max Heap gets the most congested road.

4. BFS spreads updates to neighbouring signals.

5. Greedy logic increases green time for top-density road.
