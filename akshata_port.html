Intelligent Traffic Density Management System

Problem:
Traffic becomes highly congested at certain roads, while others are empty. Manual monitoring fails during peak hours, leading to delays and slow EV movement.

Solution:
A real-time automatic system that collects traffic density, sorts it, identifies the busiest roads, and adjusts traffic signals intelligently.

Data Structures Used:

  • Array ‚Üí stores live density
  • Max Heap (Priority Queue) ‚Üí fetch highest congestion
  • Queue ‚Üí simulate vehicles waiting at signal
  • Graph ‚Üí represents connected intersections

Algorithms Used:

  • Merge Sort ‚Üí sort density values
  • Greedy Algorithm ‚Üí assign green time
  • BFS ‚Üí spread updates across road network

Step-by-Step Process:

  • Sensor counts are stored in an array.
  • Merge Sort arranges roads by traffic density.
  • Max Heap picks the most congested road.
  • BFS updates nearby intersections.
  • Greedy logic increases green time for high-density road.
  • Signals update automatically.

C++ SAMPLE CODE

#include 
using namespace std;

int main() {
    int density[] = {45, 80, 20, 62, 10};
    int n = 5;

    // Merge Sort
    auto mergeSort = [&](auto&& self, int arr[], int l, int r) -> void {
        if (l >= r) return;
        int mid = (l + r) / 2;
        self(self, arr, l, mid);
        self(self, arr, mid + 1, r);

        int temp[100], i = l, j = mid + 1, k = 0;
        while (i <= mid && j <= r)
            temp[k++] = (arr[i] < arr[j]) ? arr[i++] : arr[j++];
        while (i <= mid) temp[k++] = arr[i++];
        while (j <= r) temp[k++] = arr[j++];

        for (int x = 0; x < k; x++) arr[l + x] = temp[x];
    };

    mergeSort(mergeSort, density, 0, n - 1);

    cout << "Sorted Density: ";
    for (int i = 0; i < n; i++) cout << density[i] << " ";
    cout << endl;

    // Max Heap Simulation
    int maxDensity = density[n - 1];
    cout << "Most congested road density: " << maxDensity << endl;

    // Greedy
    int greenTime = 20 + maxDensity / 10;
    cout << "Allocated Green Time: " << greenTime << " seconds" << endl;
}
