Module 3: Loan Distribution – Case Study

Problem Statement

Banks often face situations where loan demand is higher than the available funds. To ensure fair distribution and maximize financial returns, a systematic approach is needed to allocate limited loan amounts to applicants based on profitability, urgency, and eligibility — without overshooting budget limits.

Case Study

Case Study:
How to distribute limited loans to applicants to maximize bank returns?

Algorithm: Knapsack / Dynamic Programming

Efficiency: O(nW)

C++ Pseudocode:
int maxLoanProfit(int W, vector& loans, vector& returns, int n){
    vector> dp(n+1, vector(W+1,0));
    for(int i=1;i<=n;i++){
        for(int w=0;w<=W;w++){
            if(loans[i-1]<=w)
                dp[i][w]=max(dp[i-1][w], dp[i-1][w-loans[i-1]]+returns[i-1]);
            else dp[i][w]=dp[i-1][w];
        }
    }
    return dp[n][W];
}
