Smart Irrigation & Water Distribution Network

Problem Statement:
Farmers often waste water because pipelines do not deliver water through the shortest or least-pressure-loss routes. This leads to unequal distribution and water scarcity.

Solution:
Use Dijkstra‚Äôs Algorithm to find the shortest and most efficient pipeline route so that water reaches all connected fields with minimal loss.

Algorithm: Dijkstra ‚Äì Shortest water flow route (O(E + V log V))

#include <iostream>
#include <vector>
#include <queue>
using namespace std;

int main(){
    vector<vector<pair<int,int>>> pipe(3);
    pipe[0]={{1,4},{2,7}};
    pipe[1]={{2,2}};

    vector<int> dist(3,1e9); 
    dist[0]=0;

    priority_queue<pair<int,int>,vector<pair<int,int>>,greater<>> pq;
    pq.push({0,0});

    while(!pq.empty()){
        auto [d,u]=pq.top(); pq.pop();
        for(auto [v,w]:pipe[u]){
            if(dist[u]+w < dist[v]){
                dist[v]=dist[u]+w;
                pq.push({dist[v],v});
            }
        }
    }

    for(int i=0;i<3;i++)
        cout << "Min water path to field " << i 
             << ": " << dist[i] << endl;
}

Conclusion: Water reaches farms using the most efficient pipeline network.

Pipeline Status: Water Flow: 70%

Crop Recommendation System

Problem Statement:
Many farmers choose crops without analyzing soil type, water availability, or climatic needs, leading to lower yield.

Solution:
Use Binary Search to quickly match soil type to suitable crops.

Algorithm: Binary Search on crop-soil database (O(log n))

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main(){
    vector<string> crops = {"Cotton", "Groundnut", "Millet", "Rice", "Wheat"};
    sort(crops.begin(), crops.end());

    string target = "Millet";
    int l=0, r=crops.size()-1, mid;

    while(l<=r){
        mid = (l+r)/2;
        if(crops[mid] == target){
            cout << target << " is recommended crop!" << endl;
            return 0;
        }
        else if(crops[mid] < target) l = mid+1;
        else r = mid-1;
    }

    cout << "Crop not found." << endl;
}

Conclusion: Farmers get the most suitable crop based on soil and climate.

Recommended Soil Type: Loamy

Smart Warehouse & Cold Storage

Problem Statement:
Manual storage tracking leads to spoilage and inventory mismatch.

Solution:
Use a Binary Search Tree (BST) for storing items in sorted order.

Algorithm: BST Insert & Traverse (O(log n))

#include <iostream>
using namespace std;

struct Node{
    int item;
    Node *left,*right;
    Node(int x){ item=x; left=right=NULL; }
};

Node* insert(Node* root, int x){
    if(!root) return new Node(x);
    if(x < root->item) root->left = insert(root->left,x);
    else root->right = insert(root->right,x);
    return root;
}

void inorder(Node* root){
    if(!root) return;
    inorder(root->left);
    cout << root->item << " ";
    inorder(root->right);
}

int main(){
    Node* root=NULL;
    root=insert(root,30);
    insert(root,10);
    insert(root,50);
    inorder(root);
}

Conclusion: Warehouse items stay sorted and easily searchable.

Farmers Marketplace Platform

Problem Statement:
Farmers struggle to find the best buyers for their crops at fair prices.

Solution:
Use Merge Sort to sort crops by price (best-priced crops shown first).

Algorithm: Merge Sort ‚Äì Sort crops by price (O(n log n))

#include <iostream>
using namespace std;

void merge(int a[], int l, int m, int r){
    int n1=m-l+1, n2=r-m;
    int L[n1], R[n2];

    for(int i=0;i<n1;i++) L[i]=a[l+i];
    for(int j=0;j<n2;j++) R[j]=a[m+1+j];

    int i=0,j=0,k=l;
    while(i<n1 && j<n2){
        if(L[i] <= R[j]) a[k++]=L[i++];
        else a[k++]=R[j++];
    }
    while(i<n1) a[k++]=L[i++];
    while(j<n2) a[k++]=R[j++];
}

void mergesort(int a[], int l, int r){
    if(l<r){
        int m=(l+r)/2;
        mergesort(a,l,m);
        mergesort(a,m+1,r);
        merge(a,l,m,r);
    }
}

int main(){
    int price[]={40,10,30,20};
    mergesort(price,0,3);
    for(int x:price) cout << x << " ";
}

Conclusion: Marketplace always shows best deals to farmers.

Smart Crop Storage & Expiry Tracking

Problem Statement:
Stored grains rot early because farmers do not track expiry or humidity properly.

Solution:
Use a Min Heap to always detect the crop with the earliest expiry date.

Algorithm: Min Heap ‚Äì Earliest expiry first (O(log n))

#include <iiostream>
#include <queue>
using namespace std;

int main(){
    priority_queue<int, vector<int>, greater<int>> expiry;

    expiry.push(5);
    expiry.push(2);
    expiry.push(9);

    cout << "Earliest expiry in days: " << expiry.top() << endl;
}

Conclusion: Farmers can prevent crop spoilage by monitoring earliest expiry.
